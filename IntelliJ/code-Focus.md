# IntelliJ 단축키 모음집_3

## Focus Serch

- ## **현재 파일에서 검색**

  - Mac : ```Command``` + ```F```
  - Window, Linux : ```Ctrl``` + ```F```


- ## **현재 파일에서 교체**

  - Mac : ```Command``` + ```R```
  - Window, Linux : ```Ctrl``` + ```R```


- ## **전체에서 검색**

  - Mac : ```Command``` + ```Shift``` + ```F```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```F```


- ## **전체에서 교체**

  - Mac : ```Command``` + ```Shift``` + ```R```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```R```


- ## **정규 표현식으로 검색**

  - Mac : ```Regex``` 체크하고 바꾸기
  - Window, Linux : ```Regex``` 체크하고 바꾸기


- ## **파일 검색**

  - Mac : ```Command``` + ```Shift``` + ```O```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```N```


- ## **메소드 검색**

  - Mac : ```Command``` + ```Alt``` + ```O```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```Alt``` + ```N```


- ## **Action 검색**

  - Mac : ```Command``` + ```Shift``` + ```A```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```A```


- ## **최근 열었던 파일 목록 보기**

  - Mac : ```Command``` + ```E```
  - Window, Linux : ```Ctrl``` + ```E```

- ## **최근 수정한 파일 목록 보기**

  - Mac : ```Command``` + ```Shift``` + ```E```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```E```
