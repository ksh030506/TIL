# IntelliJ 단축키 모음집_3

## Focus Serch

- ## **현재 파일에서 검색**

  - Mac : ```Command``` + ```F```
  - Window, Linux : ```Ctrl``` + ```F```


- ## **현재 파일에서 교체**

  - Mac : ```Command``` + ```R```
  - Window, Linux : ```Ctrl``` + ```R```


- ## **전체에서 검색**

  - Mac : ```Command``` + ```Shift``` + ```F```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```F```


- ## **전체에서 교체**

  - Mac : ```Command``` + ```Shift``` + ```R```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```R```


- ## **정규 표현식으로 검색**

  - Mac : ```Regex``` 체크하고 바꾸기
  - Window, Linux : ```Regex``` 체크하고 바꾸기


- ## **파일 검색**

  - Mac : ```option``` + ```Up&Down```
  - Window, Linux : ```Ctrl``` + ```W``` => 위


- ## **포커스 뒤로/앞으로 가기**

  - Mac : ```Command``` + ```Shift``` + ```O```
  - Window, Linux : ```Ctrl``` + ```Shift``` + ```N```


- ## **멀티 포커스**

  - Mac : ```option * 2``` + ```Down```
  - Window, Linux : ```Ctrl``` + ```Q```


- ## **오류 라인 자동 포커스**

  - Mac : ```F2```
  - Window, Linux : ```F2```
